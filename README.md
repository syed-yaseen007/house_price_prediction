🏠 Hyderabad House Price Prediction
This project predicts house prices in Hyderabad based on various property features using Machine Learning. It includes an interactive Streamlit web app where users can enter property details and get an instant price prediction.


🔗 Live App: https://syed-yaseen007-house-price-prediction-main-hnmqsx.streamlit.app/


📌 Features

Predicts house prices based on user-provided details.

User-friendly interface built with Streamlit.

Data preprocessing & feature engineering for improved accuracy.

Model trained with Scikit-learn.

🛠️ Tech Stack
Python

Pandas, NumPy for data processing

Scikit-learn for ML model training

Matplotlib / Seaborn for data visualization

Streamlit for app deployment
